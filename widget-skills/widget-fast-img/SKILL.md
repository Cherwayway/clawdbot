# widget-fast-img

Fast IMG Generation - Quick image generation

## Usage

```bash
widget-fast-img [options] "Text prompt for image generation"
```

## Options

- `--prompt`: Text prompt for image generation
- `--width`: Image width (default 512)
- `--height`: Image height (default 512)

## Examples

```bash
# Basic usage
widget-fast-img "your input here"

# Save to file
widget-fast-img -o output.png "your input here"
```
