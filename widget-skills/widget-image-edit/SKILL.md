# widget-image-edit

Image Edit - Edit images with prompts

## Usage

```bash
widget-image-edit [options] "Input image URL"
```

## Options

- `--image`: Input image URL
- `--prompt`: Edit instruction
- `--negative_prompt`: What to avoid

## Examples

```bash
# Basic usage
widget-image-edit "your input here"

# Save to file
widget-image-edit -o output.png "your input here"
```
