# widget-html2img

HTML2Image - Render HTML to image

## Usage

```bash
widget-html2img [options] "HTML content to render"
```

## Options

- `--html_str`: HTML content to render
- `--width`: Image width (default 800)
- `--height`: Image height (default 600)

## Examples

```bash
# Basic usage
widget-html2img "your input here"

# Save to file
widget-html2img -o output.png "your input here"
```
