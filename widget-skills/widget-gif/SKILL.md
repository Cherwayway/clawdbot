# widget-gif

GIF Generation - Create animated GIFs

## Usage

```bash
widget-gif [options] "Animation description"
```

## Options

- `--prompt`: Animation description
- `--negative_prompt`: What to avoid
- `--width`: Width
- `--height`: Height

## Examples

```bash
# Basic usage
widget-gif "your input here"

# Save to file
widget-gif -o output.mp4 "your input here"
```
