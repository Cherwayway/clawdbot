# widget-age-transform

Age Transformation - Change face age

## Usage

```bash
widget-age-transform [options] "Face image URL"
```

## Options

- `--image`: Face image URL
- `--target_age`: Target age: default, 0, 10, 20, 30, 40, 50, 60, 70, 80

## Examples

```bash
# Basic usage
widget-age-transform "your input here"

# Save to file
widget-age-transform -o output.png "your input here"
```
