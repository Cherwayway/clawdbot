# widget-social-download

Social Media Downloader - Download from YouTube/TikTok/Instagram

## Usage

```bash
widget-social-download [options] "Video URL to download"
```

## Options

- `--video_url`: Video URL to download
- `--download_type`: video or audio
- `--video_quality`: 360, 480, 720, 1080

## Examples

```bash
# Basic usage
widget-social-download "your input here"

# Save to file
widget-social-download -o output.mp4 "your input here"
```
